---

## Requirements

- ffmpeg

## Homebrew (macOS / Linux)

```bash
brew install junara/tap/encfixture
```

## Go

```bash
go install github.com/junara/encfixture@latest
```

## Binary

Download the binary for your OS from [Releases](https://github.com/junara/encfixture/releases).
